 - Python (https://www.python.org/)
 - Tkinter 
   - Windows:
     - \>>> C:\python
     - \>>> import tkinter
     - \>>> tkinter._test()
   - Linux
     - Para instalar o Python
       - $sudo apt-get install python3
     - Para instalar o Tkinter
       - $sudo apt-get install python3-tk
     - Para testar no terminal
       - $python3 -m tkinter
